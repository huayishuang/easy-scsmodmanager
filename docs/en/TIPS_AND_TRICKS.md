# 💡 Tips & Tricks

Get the most out of Easy SCSModManager.

---

## 🔍 Filter by source

The **Source** dropdown (All / Workshop / Local) is the fast way to see only
your local mods or only Workshop ones. Combined with **Ctrl+A** it makes batch
actions painless.

## 🧹 Clean out local clutter in one go

Set **Source: Local**, press **Ctrl+A** to select every visible card, then
**Delete**. Only local mods are trashed; a confirmation warns you if any are
still used by a saved profile.

## 🖱️ Move a whole selection to a group

Select several active mods (Ctrl/Shift+Click), right-click, **Move to** -> a
group. They all move at once, keeping their relative order. The **Automatic
(own category)** entry sends them back to their natural group.

## 🎯 Drag decides the group

Dropping a mod into a load-order section pins it there - the same as the
right-click move. Dragging a mod back to its home section clears the pin, so you
never accumulate stale pins.

## 🟡🔴 Read conflicts at a glance

A **⚠ yellow** glyph means the mod still does part of its job; a **⊘ red** glyph
means it is fully overwritten where it sits and you should move it up or drop it.
Hover for the exact files and winners.

## 🖥️ Use the whole window

Maximise the window: the grid adds columns to fill the width instead of leaving
an empty strip. Card size stays the same.

## 💾 Back up before big changes

Use **Backup** in the profile header before reshuffling a large load order. The
zip lives outside the game's tree, so restoring is safe.
