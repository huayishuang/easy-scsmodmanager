# ⌨️ Tastenkürzel

Alle Tastenkürzel in Easy SCSModManager.

---

## Global

| Kürzel | Aktion | Hinweis |
|--------|--------|---------|
| `F5` | Mods neu scannen | Liest `mod/`-Ordner und Workshop neu |
| `Strg+Q` | Beenden | |

## Mod-Raster (Bibliothek)

| Kürzel | Aktion | Hinweis |
|--------|--------|---------|
| `Strg+A` | Alle sichtbaren Karten wählen | Berücksichtigt den Filter (z.B. nur Lokal) |
| `Strg+Klick` | Eine Karte zur Auswahl hinzufügen/entfernen | |
| `Shift+Klick` | Bereich wählen | |
| `Entf` | Gewählte lokale Mods löschen | In den Papierkorb; Workshop-Mods übersprungen |
| `Doppelklick` | Mod aktivieren | Landet in seiner Load-Order-Gruppe |

## Aktiv-Liste

| Kürzel | Aktion | Hinweis |
|--------|--------|---------|
| `Doppelklick` | Mod aus der Aktiv-Liste entfernen | |
| `Strg/Shift+Klick` | Zeilen mehrfach wählen | Für "Verschieben nach" mit mehreren Mods |

> Tipp: Kombiniere den Filter **Quelle: Lokal** mit `Strg+A` und `Entf`, um in
> einem Rutsch aufzuräumen.
