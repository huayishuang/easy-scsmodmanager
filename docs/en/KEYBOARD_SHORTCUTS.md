# ⌨️ Keyboard Shortcuts

All keyboard shortcuts in Easy SCSModManager.

---

## Global

| Shortcut | Action | Notes |
|----------|--------|-------|
| `F5` | Rescan mods | Re-reads the `mod/` folder and Workshop |
| `Ctrl+Q` | Quit | |

## Mod grid (library)

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl+A` | Select all visible cards | Respects the current filter (e.g. only Local) |
| `Ctrl+Click` | Add / remove one card from the selection | |
| `Shift+Click` | Select a range | |
| `Delete` | Delete the selected local mods | Moves them to the trash; Workshop mods are skipped |
| `Double-click` | Activate the mod | Drops it into its load-order group |

## Active list

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Double-click` | Remove the mod from the active list | |
| `Ctrl/Shift+Click` | Multi-select rows | For a multi-mod "Move to" |

> Tip: combine the **Source: Local** filter with `Ctrl+A` and `Delete` to clear
> out local clutter in one go.
